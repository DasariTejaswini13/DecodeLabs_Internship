# Week 1 Project - Personal Portfolio Website

## Description
This project is a personal portfolio website created using HTML, CSS, and JavaScript.  
It showcases basic frontend development skills and presents personal information in a structured and responsive way.

---

## Features
- Home section with introduction
- About section describing personal details
- Skills section to show technical skills
- Projects section
- Contact section with basic form
- Responsive and clean UI design

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## How to Run
1. Open the Week1 folder in VS Code
2. Right-click on `index.html`
3. Click **"Open with Live Server"**

---

## Purpose
The purpose of this project is to practice frontend development and understand how to structure a personal portfolio website.
