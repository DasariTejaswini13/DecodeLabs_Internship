
# WEEK 2-Backend API 🚀

- This is a simple Backend API built using Node.js and Express.js.  
It demonstrates how to create a server, handle routes, and manage GET and POST requests with JSON data.

---

## 🚀 Features

- Built a server using Express.js
- Used middleware to handle JSON data (`express.json()`)
- Created GET route `/` for welcome message
- Created GET route `/profile` to return JSON data
- Created POST route `/profile` to accept and return request data
- Added basic validation for required fields
- Handled error response using HTTP status codes

---

## 🛠️ Technologies Used

- Node.js
- Express.js

---

## 🎯 Purpose

To learn backend development fundamentals including:
- Server creation
- Routing
- API handling
- JSON request/response flow

---

## 📌 How to Run

- npm install
- node server.js
